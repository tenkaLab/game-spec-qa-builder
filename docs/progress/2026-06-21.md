# 進捗ログ

Game Spec Q&A Builder の進捗を記録します。

進捗報告では、最低限以下を記載します。

- 今回追加したもの
- 方針
- 動作確認の有無
- 次の作業

運用方針:

- `progress.md` は追記方式で運用する
- 新しい進捗を上に追加し、古い進捗は下に残す
- 見づらくなった場合は、md内リンクによる目次追加、または `docs/progress/` 配下への月別・フェーズ別分割を検討する
